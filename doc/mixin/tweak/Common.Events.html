~ (void) on(element, type, callback)

Adding Dom Event

Parameters:

  • element ( DomElement, String ) — A DomElement object to apply event to, or if using a selector engine pass a string with the selector based query (Selects object based on the el property of the view)
  • type ( String ) — The type of event (For example "click")
  • callback ( Function ) — The callback function

~ (Boolean) on(name, callback, maxCalls)

Adding event from the Event API

Parameters:

  • name ( String ) — The event name, split on the / and : characters, to add
  • callback ( Function ) — The callback function; if you do not include this then all events under the name will be removed
  • maxCalls ( Number ) — The maximum amount of calls the event can be triggered.

Returns:

  • ( Boolean ) — Returns whether the event is added
