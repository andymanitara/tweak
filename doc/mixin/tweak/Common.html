~ (Object) findModule(paths, module, surrogate = null)

Try to find a module by name in multiple paths. If there is a surrogate, then if not found it will return this instead

Parameters:

  • paths ( Array<String> ) — An Array of Strings, the array contains paths to which to search for objects. The lower the key value the higher the piority
  • module ( String ) — The name of the module to search for
  • surrogate ( Object ) — (Optional) A surrogate Object that can be used if there is no module found.

Throws:

  • ( ? ) — When an object cannot be found and no surrogate is provided the following error message will appear - "Could not find a default module (#{module name}) for component #{component name}"
  • ( ? ) — When an object is found but there is an error during processing the found object the following message will appear - "Found module (#{Module Name}) for component #{Component Name} but there was an error: #{Error Message}"

Returns:

  • ( Object ) — Returns an Object that has the highest piority.
