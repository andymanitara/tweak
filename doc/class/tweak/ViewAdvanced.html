Class: tweak.ViewAdvanced

Defined in: app\lib\view_advanced.coffee
Inherits: tweak.View

Overview

The view is the DOM controller. This should be used for code that doesnt really control any logic but how the view is displayed. For example animations. The view uses a templating engine to provide the html to the DOM. The view in common MV* frameworks is typically used to directly listen for model changes to rerender however typically this should be done in the controller. The data in the model is passed into the views template, allowing for easy manipulation of the view.
