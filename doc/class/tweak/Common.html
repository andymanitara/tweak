# (Object) findModule(contexts, module, surrogate = null)

Try to find a module by name in multiple paths. A final surrogate if available will be returned if no module can be found.

Parameters:

  • paths ( Array<String> ) — An array of context paths.
  • module ( String ) — The module path to convert to absolute path; based on the context path.
  • surrogate ( Object ) — (Optional) A surrogate Object that can be used if there is no module found.

Throws:

  • ( ? ) — When an object cannot be found and no surrogate is provided the following error message will appear - "Could not find a default module (#{module name}) for component #{component name}"
  • ( ? ) — When an object is found but there is an error during processing the found object the following message will appear - "Module (#{path}) found. Encountered #{e.name}: #{e.message}"

Returns:

  • ( Object ) — Returns an Object that has the highest piority.

# (Object) require(context, module)

Require method to find a module in a given context path and module path. The context path and module path are merged together to create an absolute path.

Parameters:

  • context ( String ) — The context path
  • module ( String ) — The module path to convert to absolute path; based on the context path

Throws:

  • ( ? ) — When module can not be loaded the following error message will appear - "Can not find path #{url}"

Returns:

  • ( Object ) — Returns required object.
